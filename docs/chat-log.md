# Лог чата (контекст проекта)

> **Для агента:** при следующем обращении прочитай этот файл для восстановления контекста проекта и истории общения.

---

## 2025-03-11

### Контекст проекта (зафиксирован пользователем)

- **Идея:** Пользователь открывает HTML-страницу, вводит кодовую фразу и получает секретный контент.
- **Зашифрованный секретный контент** хранится в пазл-файле `secret.json`.
- **Подробности** по файлам и работе приложения — в `README.md`.
- **Скрипты:** `tools/encode.js` (шифрование пазл-файла), `tools/decode.js` (расшифрование).
- **Тесты** проверяют работу этих скриптов.
- **Папка `html-static/`** — HTML-приложение и тестовый пазл-файл.
- **Папка `input/`** — примеры секретных файлов; скрипт `scripts/encode_secret.sh` генерирует из них пазл-файл и складывает в `html-static/`.
- **Скрипт `scripts/decode_secret.sh`** берёт `secret.json` и раскодирует секретный контент в тестовые файлы выхода.

### Действия в чате

- Пользователь попросил вести лог общения в папке проекта и выбрать папку для чтения в следующий раз.
- Выбрана папка **`docs/`**, файл **`docs/chat-log.md`** — сюда пишется лог; при следующем сеансе агент читает этот файл для контекста.
